    • Referal System
    upon user login in the home page the the user will get his/her unique refreral link.If any one Signup with that link then the parent user will 20% of the life time earnings of the referal.However the referal commission may be limited by the admin.By navigating to > admin > settings > referal Commision it is that simple.No need to edit any code.


    Email Verification
    The Admin has the power to decide weather the new user who signup's need to verified by email or not. To make the preferences the admin should navigate to > admin > settings > Enable Email verification for new users ?


    Payout Request
    For a user to make request for payout he must satisfy the two conditions:
  • Minimum payout amount should be earned by the user.
  • Update Paypal Email in his/her account
    • However the minimum Payout will be restricted by the Admin by navigating to Admin > settings > minimum payout.


    Paypal Integration
    When a user satisfies the conditions stated above and requests for payout the payout requests will be updated in the admin section in the payout requests area.
    now admin can make payout by just clicking the paynow button and the admin will redirected to Paypal official site and make payment for users.

    Payment Proofs
    On successfull payment to user by the admin.the payment proof page and the payment history page will automatically get updated.No manual work required.
